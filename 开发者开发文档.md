# 零核服务器 - 开发者开发文档

## 项目结构

```
Zero-Core-Server/
├── data/                 # 数据存储目录
│   ├── server.json       # 主数据库文件
│   ├── secret.json       # 加密密钥文件
│   └── user/             # 用户数据目录
│       └── [UUID]/       # 每个用户的独立目录
├── views/                # EJS模板文件
│   ├── admin/            # 管理面板页面
│   ├── auth/             # 认证页面
│   ├── forum/            # 论坛页面
│   ├── partials/         # 公共组件
│   └── modules/          # 模块页面
├── public/               # 静态资源
│   ├── css/              # 样式文件
│   ├── js/               # JavaScript文件
│   └── images/           # 图片资源
├── module/               # 模块目录
├── encryption.js         # 高级加密模块
├── server.js             # 主服务器文件
├── package.json          # 项目配置
└── README.md             # 项目说明
```

## 技术栈

- **后端**: Node.js + Express.js
- **模板引擎**: EJS
- **数据库**: JSON文件存储
- **前端**: HTML/CSS/JavaScript
- **样式**: CSS自定义属性（CSS Variables）
- **加密**: Node.js内置crypto模块 + AdvancedEncryption模块
- **文件上传**: Multer
- **实时通信**: Socket.IO

## 中间件

### 认证中间件
```javascript
function requireAuth(req, res, next) {
  if (!req.session.userId) {
    return res.redirect('/login');
  }
  next();
}

function requireAdmin(req, res, next) {
  const user = getUserBySession(req);
  if (!user || (user.author !== 'admin' && user.author !== 'system')) {
    return res.status(403).send('权限不足');
  }
  next();
}

function requireSystem(req, res, next) {
  const user = getUserBySession(req);
  if (!user || user.author !== 'system') {
    return res.status(403).send('权限不足');
  }
  next();
}
```

### 文件上传中间件
```javascript
const upload = multer({
  storage: storage,
  limits: { fileSize: 2 * 1024 * 1000 }, // 限制2MB
  fileFilter: (req, file, cb) => {
    if (file.mimetype.startsWith('image/')) {
      cb(null, true);
    } else {
      cb(new Error('只允许上传图片文件'), false);
    }
  }
});
```

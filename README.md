# 零核服务器 (Zero-Core-Server)

> **"为社区而生的轻量级用户管理系统"**

零核服务器是一个基于 Node.js 的现代化用户管理系统，提供用户认证、权限管理、模块扩展、论坛交流等丰富功能。采用 Express + EJS + Socket.IO 技术栈，支持多视图引擎和沙盒隔离的模块系统，让扩展功能变得简单安全。

## ✨ 核心功能

### 用户系统
- 🔐 用户注册/登录（支持用户名密码和 UUID 登录）
- 👤 个人资料管理（头像、背景图、个人简介）
- 🎨 主题切换（深色/浅色模式）
- 💰 积分系统（每日登录奖励）
- 🔑 多级权限管理（user/admin/system）

### 论坛系统
- 📝 发帖/回帖功能
- 📂 多分类讨论区
- 📝 Markdown 格式支持

### 模块系统
- 🧩 沙盒隔离的模块运行环境
- 🎨 多视图引擎支持（EJS/Handlebars/Pug/Mustache）
- ⭐ 模块收藏功能
- 💬 模块评论功能
- 🔍 模块搜索功能

### 系统管理
- 🛠️ 用户管理（创建/编辑/删除）
- 📊 系统状态监控
- 🔐 多层加密密钥管理
- 📢 系统公告管理

## 🚀 快速开始

### 环境要求
- Node.js 14.x 或更高版本
- Windows/Linux/macOS 操作系统

### 安装步骤
```bash
# 克隆项目
git clone <repository-url>
cd Zero-Core-Server

# 安装依赖
npm install

# 启动服务器
node server.js
```

### 访问服务
服务器默认运行在 `http://localhost:5000`

## 📁 项目结构

```
Zero-Core-Server/
├── data/                 # 数据存储目录
│   ├── server.json       # 主数据库
│   ├── secret.json       # 加密密钥
│   ├── user/             # 用户数据
│   └── module/           # 模块数据
├── views/                # 视图模板
├── public/               # 静态资源
├── module/               # 模块目录
├── encryption.js         # 加密模块
├── module-sandbox.js     # 模块沙盒
└── server.js             # 主服务器
```

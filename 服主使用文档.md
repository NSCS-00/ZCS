# 零核服务器 - 服主使用文档

## 概述

零核服务器是一个功能完整的Node.js用户管理系统，支持多用户、权限管理、积分系统、论坛功能和个人简介等功能。本文档将指导您如何部署、配置和管理服务器。

## 部署指南

### 环境要求
- Node.js 14.x 或更高版本
- Windows/Linux/macOS 操作系统
- 至少512MB可用内存

### 安装步骤
1. 下载项目文件到服务器目录
2. 在项目根目录执行：
   ```bash
   npm install
   ```
3. 启动服务器：
   ```bash
   node server.js
   ```
   或使用PM2进行进程管理：
   ```bash
   pm2 start server.js --name "zero-core-server"
   ```

### 端口配置
- 默认端口：5000
- 如需修改端口，编辑`server.js`文件中的`PORT`变量

## 系统管理

### 用户管理
#### 普通用户 (user)
- 基本功能使用权限
- 可修改个人信息和头像
- 每日登录获得5-10积分奖励
- 可设置个人主题偏好
- 可创建个人简介页面

#### 管理员 (admin)
- 可管理普通用户
- 无法管理同级或更高级用户
- 可访问管理面板
- 可查看系统统计信息
- 可创建、修改、删除用户（不能操作同权/更高权限用户）

#### 系统管理员 (system)
- 拥有最高管理权限
- 可管理所有用户
- 可执行系统控制操作（重启、关闭服务器）
- 可分配管理员权限给其他用户
- 可管理加密密钥

### 管理面板功能

#### 总览页面
- 查看系统访问人次
- 统计用户数量
- 监控服务器CPU、内存使用情况
- 查看系统负载信息

#### 用户管理
- 查看所有用户信息（用户名、UUID、权限、积分、注册时间等）
- 创建新用户
- 修改用户信息（用户名、权限、积分、头像等）
- 删除用户账户
- 注意：admin权限用户无法修改同级或更高级用户

#### 系统控制（仅system权限）
- 重启服务器
- 关闭服务器
- 系统公告管理
- 加密密钥管理

### 系统公告管理
- 通过管理面板的系统控制页面可以更新系统公告
- 公告支持Markdown格式
- 所有用户都可以在主页看到系统公告

### 加密密钥管理
- 主密钥：系统最高级密钥
- 副密钥：用主密钥加密
- 系统密钥：用副密钥加密
- 文件密钥：用系统密钥加密
- 密钥会自动轮换以增强安全性

## 数据管理

### 用户数据存储
- 用户数据存储在 `./data/server.json`
- 用户文件夹位于 `./data/user/[UUID]`
- 每个用户有自己的文件夹存储头像、背景图、个人简介等文件

### 备份策略
定期备份以下目录：
- `./data/` - 包含所有用户数据
- `./public/uploads/` - 包含上传文件

## 性能监控

### 资源使用
- 通过管理面板监控CPU和内存使用
- 建议保持内存使用率低于80%

### 访问统计
- 系统自动记录访问人次
- 可通过管理面板查看统计数据

## 安全配置

### 权限控制
- 严格遵循权限等级制度
- admin无法修改同级或更高级用户
- 敏感操作需要相应权限

### 数据安全
- 用户密码使用多层加密
- 定期更新加密密钥
- 限制对数据文件的访问权限

## 功能特色

### 个人简介系统
- 每个用户可以创建个人简介页面
- 支持背景图上传
- 支持简单的Markdown格式

### 论坛系统
- 多分类讨论区
- 支持发帖和回帖
- Markdown格式支持

### 主题系统
- 支持深色/浅色主题切换
- 用户可自定义主题偏好
- 系统会记住用户的选择

### 模块系统
- 支持UI模块和API模块
- 模块中心便于管理
- 支持模块热插拔

## 故障排除

### 服务器无法启动
1. 检查Node.js版本是否符合要求
2. 确认端口未被占用
3. 检查依赖包是否完整安装
4. 检查数据库文件权限

### 用户无法登录
1. 检查用户名和密码是否正确
2. 确认用户账户状态
3. 检查数据库文件权限

### 主题设置不生效
1. 清除浏览器缓存
2. 检查用户数据中的主题设置

### 模块功能异常
1. 确认模块配置文件格式正确
2. 检查模块文件权限

## 维护建议

### 定期维护
- 每周备份数据文件
- 监控服务器资源使用
- 检查系统日志
- 清理过期的临时文件

### 性能优化
- 定期清理无用的用户文件
- 监控数据库文件大小
- 优化静态资源缓存

## 常见问题

### 如何添加管理员账户？
1. 使用system权限账户登录
2. 进入管理面板 → 用户管理
3. 编辑目标用户，将其权限改为admin

### 如何重置用户密码？
1. 使用具有管理权限的账户登录
2. 进入管理面板 → 用户管理
3. 直接修改目标用户的密码

### 服务器响应慢怎么办？
- 检查服务器资源使用情况
- 优化数据库查询
- 考虑升级服务器配置
- 检查是否有大量用户数据

### 如何管理加密密钥？
- 通过管理面板的系统控制页面
- 可更新主密钥、副密钥和系统密钥
- 系统会自动重新加密相关数据

## 升级指南

### 版本升级
1. 备份现有数据
2. 下载新版本代码
3. 安装新依赖（如有）
4. 迁移旧数据（如有需要）
5. 启动新版本服务器

### 数据迁移
- 系统会自动处理大部分数据格式变更
- 重要变更会在更新日志中说明
- 建议升级前备份数据

## 安全建议

### 访问安全
- 使用强密码
- 定期更换管理员密码
- 限制服务器访问IP
- 启用HTTPS（推荐反向代理）

### 数据安全
- 定期备份数据
- 保护数据库文件权限
- 监控异常访问日志
- 定期更新加密密钥
